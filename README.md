# MemoChamp

Un jeu simple et ludique pour entraîner la mémoire à court terme, inspiré du concept du **Dual N-Back**.  
🎯 Objectif : retenir l’emplacement de carrés bleus dans un temps imparti.

---

## 🔗 Démo

👉 [Voir le jeu en ligne](https://threefrogs511.github.io/memochamp/)

---

## 🛠 Technologies utilisées

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage
- Fichier JSON statique (`scoreTemplate.json`)

## 🧠 Fonctionnalités

- Affichage de scores
- Possibilité de modifier le temps à sa guise
- Enregistrement des données localement via `localStorage`  
- Lecture d’un fichier JSON pour charger les scores par défaut  
- Gestion de plusieurs niveaux de difficulté  
- Interface simple, responsive et accessible

---

## 🎯 Objectif du projet

Ce projet a été conçu pour pratiquer les bases du développement frontend avec JavaScript, tout en développant un jeu interactif autour de l'entraînement cognitif.
